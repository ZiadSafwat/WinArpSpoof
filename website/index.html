Windows ARP CLI Tool

A powerful command-line utility for network administrators.

Download Now

Key Features

  • Scan Network

    Quickly discover devices on your network.

  • Block IP

    Prevent specific devices from communicating.

  • Unblock IP

    Restore communication to blocked devices.

  • ARP Spoofing Protection

    Protect against ARP spoofing attacks.

Usage Examples


arpcli scan
arpcli block 192.168.1.100
arpcli unblock 192.168.1.100
arpcli start
arpcli stop
arpcli interfaces
